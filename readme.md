## Tech

- **Eleventy** static site generator
- Based on the [Eleventy Excellent](https://github.com/madrilene/eleventy-excellent) starter (by Lene Saile), which follows the approach from [buildexcellentwebsit.es](https://buildexcellentwebsit.es/)
- Tailwind CSS, image optimization (Eleventy-img), syntax highlighting, RSS, dark/light theme

## Run locally

```bash
npm install
npm start
```

Production build:

```bash
npm run build
```
